关于博客建立的若干阻碍

发表于 2019-11-08 分类于 普通问题

博客部署GitHub时出现404

情况一

仓库名、GitHub用户名、Git用户名不一致。
把各个名称统一即可。
如：github ID是：abc123 

仓库名应该是abc13.github.io  
本地git的用户名 在配置时也应该是 abc13

git 本地配置用户名方法 ：
启动git/bin/bash 运行如下命令
git config –global user.name “username”
git config –global user.email “email”

情况二

仓库名、GitHub用户名、Git用户名一致。依然出现404页面无法找到
我的解决办法。
断网后重连/从无线网切换的有线网。
